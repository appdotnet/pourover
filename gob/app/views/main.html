PourOver from App.net

PourOver is a tool that allows you to publish any RSS feed to App.net.

  • Step 1 Click the Authorize with App.net button below
  • Step 2 Sign up or Log in
  • Step 3 Enter your RSS feed

Authorize with App.net

It's an easy to way to syndicate whatever you are publishing to the growing community of App.net users. Getting started is easy, even if you don't yet have an App.net account. Learn more about App.net.

Some examples:

Options

Use Linked List Mode if you run a link blog and you want the title to be linked to your website instead of the post permalink.

If PourOver can find an image in an feed item it will be attached to a post as an oEmbed annotation. It uses the following methods: Media RSS and first image in the body of the post. Dimensions Min (200 x 200 px) Max (1000 x 1000 px).
