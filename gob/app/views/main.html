How do you want the title and link to show up in your App.net posts

Enter your Bit.ly credentials to turn on link shortening. Find your API Key information.

Include a portion of the feed item body.

If an image is included in the feed item it will be attached to the App.net post as an oEmbed annotation. To be included the image must specify its width and height. Dimension constraints: Min (200 x 200 px) Max (1000 x 1000 px).

If a video(Youtube/Vimeo) is embedded in the feed item it will be attached to the App.net post as an oEmbed annotation. If both include video, and include image are checked include video will win.

Use Linked List Mode if you run a link blog and you want the title to be linked to your website instead of the post permalink.
